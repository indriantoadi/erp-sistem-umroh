# Modul Sistem Pembayaran & Operasional - ERP Umroh

Repositori ini berisi modul inti dari sistem **Enterprise Resource Planning (ERP)** yang saya rancang untuk operasional travel umroh dan haji. Modul ini berfokus pada otomasi pemrosesan data jamaah, pengelolaan cabang, dan pelaporan keuangan.

## 🛠️ Fitur Utama
- **BookingController:** Menangani logika alur registrasi jamaah dan manajemen data *booking* secara terstruktur.
- **BranchController:** Mengelola data dan sinkronisasi operasional antar-cabang agensi travel.
- **ReportController:** Memproses data transaksi untuk menghasilkan laporan keuangan yang akurat dan siap audit.

## 💻 Tech Stack
- **Framework:** Laravel
- **Language:** PHP
- **Database Architecture:** MySQL

## 🚀 Highlights
- **Struktur Kode:** Menggunakan pola desain MVC (*Model-View-Controller*) untuk memastikan kode tetap bersih dan mudah dipelihara.
- **Integrasi Data:** Modul ini dirancang untuk meminimalkan *human error* dalam rekonsiliasi data keuangan dan pelaporan operasional.
